# VS-Quote for VS Code

A beautifully simple, ultra-lightweight extension that provides an hourly dose of inspiration directly in your VS Code status bar.

## Features

* **Ambient Inspiration:** Displays a short snippet of an inspirational quote in the bottom right corner of your editor.
* **Hover for More:** Simply hover your mouse over the status bar item to read the full quote and the author's name.
* **Zero Distractions:** Fully unobtrusive. No annoying pop-ups, no clickable alerts, and no heavy UI. Just clean text that updates in the background.
* **Automatic Updates:** Fetches a fresh quote automatically every hour without needing to restart the editor.

## How it Works

This extension uses the free [ZenQuotes API](https://zenquotes.io/) to pull a new quote each hour. It runs a lightweight background check periodically to ensure your quote is always up to date. 

## Installation

1. Open VS Code.
2. Go to the Extensions view (`Ctrl+Shift+X` or `Cmd+Shift+X`).
3. Search for `VS-Quote`.
4. Click **Install**.
5. The quote will appear immediately in your status bar!

## Requirements

An active internet connection is required to fetch the quotes. If you are offline, the status bar will gracefully display a "Quote offline" indicator until your connection returns.
